Before you go on
As I'm still a rookie in programming, please don't expect world class websites, applications and perfect code yet. I just wanted to share here some of my first experiments which I'm proud I could create. I didn't include stuff here which I wrote during a course following an instructor, but rather all you can see here below I made them work all on my own.

Let's check them out!

All-in-one startpage

HTML CSS Bootstrap PHP MySQL JavaScript jQuery

A single page web app that contains all my most frequently used bookmarks and functionalities.

Welcome message

Stores user name and gets location and weather report by IP address

Search boxes

In addition to Google I can search from all my bookmarks stored in a MySQL database or look up words in my Korean word list.

Notes

Notes app with functions like add new, update, archive or delete a note.

Link tiles

My most frequently used links in scrollable boxes with cool hover effects.

Ptkin.net

HTML CSS Bootstrap JavaScript jQuery PHP MySQL

My first and biggest project yet is this website you're browsing now. I started to work on it during a basic web developer course, and since then I'm always experimenting on it with new techniques I get to know over time. Some of the cool features I implemented and I'd like to mention are the followings:

Responsive design

It's fully responsive thanks to Bootstrap, so all pages should look fine on both desktops, tablets and mobile phones.

Animated elements

To make the site a bit more dynamic, I added some animations using both Bootstrap and jQuery.

Contact form

If you want to send a message to me there is a contact form available at the bottom using the mail functions of PHP.

My own Korean-Hungarian dictionary

My favorite and most useful feature is the dictionary with search function (with autocomplete) that uses my own word list as resource. It's all powered by JavaScript and a bit of Python to convert the Excel word list to a format that JS can use.

Connection to a database

The Links page uses AJAX calls and PHP to fetch data from my SQL database. The search function with autocomplete is also available here.

Password validation

If you want to download files from my webiste, you'll need a password which I validate with PHP and hashing.

MySQL database admin page

HTML Bootstrap PHP MySQL JavaScript jQuery

Tailor made admin page to easily access tables and manipulate records in a MySQL database using PHP and JavaScript AJAX calls. For the design of the page I only used simple Bootstrap styling.

Basic functionality

Exactly those functions which are needed: search, update, delete records, and export table to a CSV file.

Access control

Anyone can view and search the database, but manipulating data is password protected.

Dictionary desktop app

Python Desktop

Not a pretty one, but it's my first working, functional and useful desktop application. All written in Python, for the back-end part I used OpenPYXL, and the front-end is created by the Tkinter library.

Read data from Excel

The program is using my own word list that is stored in an Excel file.

Ability to add new words

Unlike my web dictionary, this app is also capable of adding and saving new word pairs to the recource file.

Node.js database API

Node.js SQLite

Basic API powered by a Node.js web server that allows a user to fetch data from an SQLite database and receive it in JSON format. It also has functionalities to manipulate the data or download the whole database.

URL routing

To search for a record or fetch a whole table it's enough to use the specific URL, like .../search/{word} or .../list/{tablename}.

Request handling

For features like updating or removing records the server needs the appropriate parameters in the request body object.

I hope you liked them!
